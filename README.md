## 构建 / 自部署

### 1. 安装依赖
```
pnpm install
```

### 2. 构建后端

```
cd backend
pnpm run build
```

### 3. 构建前端

```
cd frontend
pnpm run build
```

### 4. 启动项目

```
cd backend
pnpm run start
```
